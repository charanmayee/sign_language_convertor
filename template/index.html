What it does

✋
Real-Time Hand Detection

MediaPipe tracks 21 hand landmarks per frame at 30fps+, enabling fluid, low-latency gesture capture through any standard webcam.

🧠
CNN Gesture Recognition

A Convolutional Neural Network trained on preprocessed hand images classifies A–Z sign language alphabets with up to 95% accuracy.

📝
Text Output

Recognized gestures are converted to readable characters in real-time, building words and sentences on screen as you sign.

🔊
Text-to-Speech

pyttsx3 converts accumulated text into audible speech on demand — press S and hear your signs spoken aloud.

♿
Accessibility First

Designed as a low-cost assistive tool requiring only a standard webcam — no specialized hardware or proprietary devices needed.

🔌
Modular Architecture

Cleanly separated modules for detection, training, and inference — easy to extend with new gesture sets or deploy to different platforms.

Setup

Get started

1
Clone the repository
git clone https://github.com/your-username/sign-language-project.git cd sign-language-project
2
Create virtual environment
python -m venv venv venv\Scripts\activate # Windows # source venv/bin/activate # macOS / Linux
3
Install dependencies
pip install -r requirements.txt
4
Train the model
python train.py # Trains CNN and saves sign_model.h5
5
Run real-time detection
python app.py # Starts webcam + live gesture recognition
Controls

Keyboard shortcuts

S
Convert accumulated text to speech via pyttsx3
C
Clear the current text buffer and start fresh
ESC
Exit the application and close webcam stream
